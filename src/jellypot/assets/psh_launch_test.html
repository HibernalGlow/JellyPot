PowerShell 启动三种方案模拟

方案一：wt.exe 无头启动

wt.exe -w 0 nt -p "Windows PowerShell" powershell -NoProfile -WindowStyle Hidden -File "d:\\1VSCODE\\Projects\\LazyCommand\\JellyPot\\potplayer.ps1" %1
需要 Windows Terminal (wt.exe) 已安装，Win10 1903+ 支持。

方案二：PowerShell 原生隐藏窗口

powershell -NoProfile -WindowStyle Hidden -File "d:\\1VSCODE\\Projects\\LazyCommand\\JellyPot\\potplayer.ps1" %1
兼容性最好，推荐！

方案三：cmd /c start 隐藏窗口

cmd /c start "" /min powershell -NoProfile -File "d:\\1VSCODE\\Projects\\LazyCommand\\JellyPot\\potplayer.ps1" %1
窗口会闪一下，适合批处理场景。
